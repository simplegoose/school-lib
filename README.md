## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

Ensure you have ruby installed in your machine. If you haven't installed ruby visit [Ruby download page](https://www.ruby-lang.org/en/downloads/)

### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git clone  https://github.com/simplegoose/school-lib.git
```

### Install

Install this project with:

```sh
  cd my-project
  gem install or bundle install
```

### Usage

To run the project, execute the following command:

```sh
  ruby teacher.rb
  ruby student.rb
```

### Run tests

To run tests, run the following command:

```sh
  bin/rails test test/models/article_test.rb
```
